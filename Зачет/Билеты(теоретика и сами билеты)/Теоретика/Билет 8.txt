БИЛЕТ 8
Элемент пользовательского интерфейса WebBrowser. 
Основное назначение. Основные свойства и методы. Приведите примеры.

Элемент управления предоставляет управляемую оболочку 
для элемента управления ActiveX WebBrowser. Управляемая оболочка 
позволяет отображать веб-страницы в клиентских приложениях Windows Forms.

Свойство Document	
Возвращает объект, предоставляющий управляемый доступ к объектной 
модели документа HTML текущей веб-страницы.

Свойство DocumentText	
Возвращает или задает содержимое HTML текущей веб-страницы.

Свойство DocumentTitle	
Возвращает заголовок текущей веб-страницы.

Метод GoBack	
Переход к предыдущей странице в журнале.

Метод GoForward	
Переход к следующей странице в журнале.

Метод Navigate	
Переход по указанному URL-адресу.

Свойство ObjectForScripting	
Возвращает или задает объект, который может использоваться кодом 
скрипта веб-страницы для взаимодействия с приложением.

Метод Print	
Выводит текущую веб-страницу.

Метод Refresh	
Перезагружает текущую веб-страницу.

Метод Stop	
Останавливает текущую навигацию и останавливает динамические элементы 
страницы, такие как звуки и анимация.

Свойство Url	
Возвращает или задает URL-адрес текущей веб-страницы. При установке этого 
свойства элемент управления перемещается на новый URL-адрес.
